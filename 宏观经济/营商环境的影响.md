###  营商环境的影响
---
```mermaid

sequenceDiagram
    participant 营商环境
    participant 企业与企业家
    participant 创新生态
    participant 经济
    participant 政府与社会

    营商环境 ->> 营商环境: 初始状态（可能不完善）
    activate 营商环境
        note over 营商环境: 营商环境初始状态可能存在障碍
      alt 营商环境不完善
        营商环境 -->> 企业与企业家: 扼杀企业家精神，损害企业长期发展
          note over 企业与企业家: 不良营商环境打击企业和企业家积极性
        营商环境 -->> 创新生态: 阻碍创新生态形成
        营商环境 -->> 经济: 经济增长受阻
      else 营商环境改善
        营商环境 -->> 企业与企业家: 激发企业家精神，促进企业长期发展
          note over 企业与企业家: 良好营商环境激励企业和企业家
        营商环境 -->> 创新生态: 促进创新生态发展
        营商环境 -->> 经济: 经济高质量发展
      end
    deactivate 营商环境

    政府与社会 ->> 营商环境: 共同承担改善责任
    activate 政府与社会
         note over 政府与社会: 政府和社会共同致力于改善营商环境
       政府与社会 ->> 政府与社会: 共同努力，构建公平、公正、透明、可预期环境
    deactivate 政府与社会

    activate 营商环境
      营商环境 ->> 营商环境: 营商环境得到改善
       note over 营商环境: 营商环境达到理想状态
    deactivate 营商环境

    营商环境 ->> 经济: 实现经济高质量发展
    note over 经济: 良好的营商环境推动经济高质量发展




```
