

```mermaid
sequenceDiagram
    participant 政府 as 政府/政策制定者
    participant 居民 as 消费者/居民
    participant 企业 as 企业/市场参与者

    %% 1. 促进收入增长
    loop 促进收入增长
        政府->>政府: 完善收入分配制度
        政府->>居民: 健全工资决定机制，提高最低工资标准
        政府->>居民: 加大低收入群体转移支付
        政府->>居民: 推进个税改革，降低中低收入税负
        政府->>居民: 研究开征遗产税、房产税等
    end

    %% 2. 健全社会保障体系
    loop 健全社会保障体系
        政府->>居民: 拓宽社会保障覆盖面（灵活就业人员）
        政府->>居民: 提高社会保障待遇水平（养老金、医保报销比例）
        政府->>居民: 提升社保管理和服务质量
    end

    %% 3. 规范房地产市场
    loop 规范房地产市场
        政府->>居民: 稳定房价，坚持“房住不炒”
        政府->>居民: 增加住房供给（保障性住房和租赁市场）
        政府->>居民: 打击投机炒房行为
    end

    %% 4. 优化消费环境
    loop 优化消费环境
        政府->>企业: 加强市场监管（打击假冒伪劣商品、价格欺诈）
        政府->>企业: 完善消费者权益保护制度
        企业->>居民: 提升服务质量，增强满意度
    end

    %% 5. 引导理性消费
    loop 引导理性消费
        政府->>居民: 加强消费教育和引导（理性消费、绿色消费）
        政府->>居民: 促进消费结构升级（服务消费、信息消费）
    end

    %% 6. 提振消费信心
    loop 提振消费信心
        政府->>居民: 保持经济平稳运行（稳定就业、控制通胀）
        政府->>居民: 加强信息公开和透明度（发布经济信息）
    end

    %% 最终目标
    居民->>企业: 增强消费能力，释放消费潜力
    居民->>企业: 提升消费体验，促进消费升级
    居民->>企业: 树立理性消费观念，提振消费信心
```
